About the Project

The Problem: Marine biofouling and the costly, environmentally impactful process of dry docking ships for hull cleaning.

Objective: To build an autonomous, eco-friendly hull cleaner robot that operates underwater to maintain hull cleanliness sustainably.

Sustainable Development Goals: This project aligns with SDG 9 (Industry, Innovation, and Infrastructure) and SDG 14 (Life Below Water).

Target Users:

  • Shipping companies
  • Navy fleets
  • Port authorities and operators

Key Features

BAHRI incorporates advanced robotics and sustainable design to provide efficient, marine-safe hull cleaning:

Eco-Cleaning System

Environmentally safe bio-cleaning technology that protects marine life.

Robotic Arms & Tool-Swapping

Adaptive robotic arms can swap tools for optimized cleaning.

Solar-Powered Dock

Self-sustaining energy system with solar power for recharging.

AI Navigation

Smart navigation using AI for efficient hull mapping and cleaning.

Sustainable Materials

Made from recycled plastic and aluminum for environmental responsibility.

Marine-Safe Operation

Operates without harmful chemicals or damage to marine ecosystems.

How It Works

1
BAHRI approaches the ship hull and performs eco-friendly cleaning using specialized brushes and suction.
2
Robotic arms swap tools as needed to handle different fouling scenarios.
3
Once cleaning is complete, BAHRI docks to the solar-powered station to recharge.
4
AI analyzes hull condition data and plans the next cleaning cycle autonomously.

// Example Arduino code snippet for controlling robotic arm
#include <Servo.h>

Servo armServo;

void setup() {
  armServo.attach(9);
  Serial.begin(9600);
}

void loop() {
  armServo.write(90);  // Move arm to neutral position
  delay(1000);
  armServo.write(45);  // Move arm to cleaning tool position
  delay(2000);
}
